[ALERT-warning]In this problem you need to create you own iterator but the theory has not given an example for it. You may skip the problem or just to google how to do that. Actually, the problem is not complicated.[/ALERT]

Given a class named Range. It represents a range from A (inclusive) to B (exclusive). The class implements the interface Iterable, therefore, an instance of Range can be used in the for-each loop.  

Write a body of the overridden method iterator so that the following code works correctly:

Range range = new Range(2, 6);
for (Long val : range) {
    System.out.println(val);
}

It must print:

2
3
4
5

Sample Input:
2 6
Sample Output:
2
3
4
5


Memory limit: 256 Mb
Time limit: 5s
